# BERTA - Painel Operacional do Supervisor

Painel de produtividade para supervisores de campo.

## Telas
- 📅 Diário - Controle do dia
- 📊 Produção Diária - Análise de produtividade
- 🔁 Repetidos - Controle de repetições 30 dias
- 👶 Infância - Instalações com reparo em 30 dias
- 📆 Calendário - Visão mensal por técnico

## Fonte de Dados
BASEBOT.csv hospedado no Supabase Storage
